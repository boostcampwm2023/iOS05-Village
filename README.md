## 🛠️ 주요 기능
- 필요한 물품이나 빌려주고 싶은 물품을 게시글로 등록할 수 있습니다.
- 사용자간 실시간 채팅 서비스를 제공합니다.
- 필요한 물품을 손쉽게 찾을 수 있는 검색 기능을 제공합니다.
